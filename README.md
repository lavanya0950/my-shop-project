🛒 My Shop Project
A simple beginner-friendly shopping cart project built using HTML, CSS, and JavaScript.
This project demonstrates basic e-commerce functionality like product listing, cart management, and total calculation.
🚀 Features
🛍️ Display products dynamically
➕ Add products to cart
🔢 Quantity support for items
❌ Remove items from cart
💰 Automatic total price calculation
🧑‍💻 Tech Stack
HTML
CSS
JavaScript (Vanilla JS)
📁 Project Structure

MyShopProject/
│
├── index.html
├── script.js
└── style.css (optional)
⚙️ How to Run
Download or clone this repository
Open the folder
Open index.html in a browser
Or use Live Server (VS Code) for better experience
